# Sajid Portfolio - GitHub Pages (p5)

This project is configured for:

https://sajidulhsajid-creator.github.io/p5/

## Important

Use GitHub Actions for Pages deployment. Do not deploy directly from the main branch, because Vite needs to build the project into the `dist` folder first.

## GitHub Pages Settings

Repository → Settings → Pages → Build and deployment → Source → GitHub Actions

## Deploy

Push these files to the root of the `p5` repository. Then run:

Actions → Deploy static content to Pages → Run workflow
